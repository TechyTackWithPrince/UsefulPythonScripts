# Celsius to Fahrenheit Converter

A simple web application built with Flask that converts Celsius temperatures to Fahrenheit degrees. This application provides a user-friendly interface for entering a temperature in Celsius and instantly calculates its equivalent value in Fahrenheit.

## Table of Contents

- [Prerequisites](#prerequisites)
- [Usage](#usage)

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Python (3.6 or higher) installed on your system.
- Flask, a Python web framework, installed. You can install it using `pip`:
```
pip install flask

or

sudo python3 -m pip install flask
```


## Usage

- Run
    ```
    python3 app.py
    ```
- Open your web browser and navigate to http://127.0.0.1:8080.
- You'll see the "Celsius to Fahrenheit Converter" interface.
- Enter a temperature in Celsius in the input field and click "Convert to Fahrenheit."
- The equivalent temperature in Fahrenheit will be displayed on the page.
